<p>ExpressVPN在Windows，Mac OS和路由器的中提供了一个可选的拆分隧道(split tunneling)功能，名字取得很晦涩，其实很简单。就是把国外和国内的流量分开，访问百度的时候不通过VPN连接，访问谷歌的时候通过VPN连接。这样，你就不用手动连接/断开，并且加快你访问国内网络的速度，这个功能还是非常人性化。</p>
